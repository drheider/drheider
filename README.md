# Machine Learning Researcher · Medical Image Analysis · On-Device AI Systems

I work on machine learning for medical image analysis, with a focus on melanoma analysis.

What I enjoy most is taking models beyond experimentation and turning them into systems that can actually be used — including on-device applications such as an iPad-based melanoma analysis app and a fully offline iOS LLM chat system.

I first explored computational intelligence in the 1990s, leading to my PhD thesis. Today, I work with modern machine learning approaches, with a strong interest in privacy-preserving, on-device AI.

---

## Focus

- Medical Image Analysis (Melanoma)
- On-Device AI Systems
- Privacy-Preserving Machine Learning
- Local Large Language Models (LLMs)

## 🧬 Research & Engineering Pipeline

### Model Architectures
*   **EfficientNet**
*   **ConvNeXt**
*   **Swin Transformer**

### Pipeline Evolution
1.  **TensorFlow:** Initial implementation.
2.  **PyTorch Native:** Migrated for a better operating system compatibility and GPU utilization.
3.  **PyTorch MONAI:** Adopted for domain-specific medical imaging capabilities with a nicer pipeline.
4.  **PyTorch Native (Current):** Reverted to a native implementation using `torchvision` transforms, mainly to integrate **RandAugment**.
