## ðŸ§¬ Research & Engineering Pipeline

### Model Architectures
*   **EfficientNet**
*   **ConvNeXt**
*   **Swin Transformer**

### Pipeline Evolution
1.  **TensorFlow:** Initial implementation.
2.  **PyTorch Native:** Migrated for a better operating system compatibility and GPU utilization.
3.  **PyTorch MONAI:** Adopted for domain-specific medical imaging capabilities with a nicer pipeline.
4.  **PyTorch Native (Current):** Reverted to a native implementation using `torchvision` transforms, mainly to integrate **RandAugment**.
